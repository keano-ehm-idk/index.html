De Stelling van Pythagoras

Leer hoe je de lengte van een zijde van een rechthoekige driehoek kunt berekenen

Wat is de Stelling van Pythagoras?

De stelling van Pythagoras is een wiskundige regel die alleen werkt bij een rechthoekige driehoek. Dat is een driehoek met één hoek van precies 90 graden.

De formule is:

a² + b² = c²

Hierbij zijn:

  • a = lengte van één rechte zijde
  • b = lengte van de andere rechte zijde
  • c = de schuine zijde (de langste zijde)

Hoe reken je het stap voor stap uit op papier?

Voorbeeld:

Stel:

  • a = 3
  • b = 4
  • c = ?

Stap 1: Schrijf de formule op.

a² + b² = c²

Stap 2: Vul de getallen in.

3² + 4² = c²

Stap 3: Reken de kwadraten uit.

9 + 16 = c²

Stap 4: Tel op.

25 = c²

Stap 5: Trek de wortel.

c = √25

Stap 6: Antwoord.

c = 5

Handige Tip

Als je een rechthoekige driehoek ziet in een som, staat er vaak een klein vierkantje bij de hoek van 90 graden.

Je kunt dit vierkantje ook zien als een pijltje dat naar de schuine zijde wijst. De zijde waar het vierkantje naartoe wijst is dus de schuine zijde (c).

Dit helpt je snel te herkennen welke zijde je moet berekenen met de stelling van Pythagoras.

Wanneer mag je Pythagoras gebruiken?

  • Alleen bij een rechthoekige driehoek
  • Als je twee zijden weet
  • Om de derde zijde te berekenen
